# Changelog

All notable changes to this project are documented here.

## 0.2.0

- Added live keyboard settings retrieval and per-key readback summaries.
- Added firmware-aware ×10, ×100, and ×200 travel conversion.
- Added guarded factory reset with reconnect and readback flow.
- Added automatic process-based Daily/Game profile switching.
- Added direct WebHID and local Windows bridge transports.
- Added production deployment configuration and automated tests.
